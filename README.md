
# 🌤️ Modern Weather Dashboard

A beautiful, fully responsive weather dashboard built with HTML, CSS, and vanilla JavaScript. It fetches live weather data using the OpenWeatherMap API and features a sleek, modern **Glassmorphism** interface with built-in Light/Dark mode support.

## ✨ Features

- **Live Weather Data:** Get real-time temperature, humidity, wind speed, and atmospheric pressure.
- **5-Day Forecast:** Displays a neat 5-day weather forecast layout.
- **Glassmorphism UI:** Modern, frosted-glass design aesthetics.
- **Light & Dark Mode:** Seamlessly toggle between themes (saves user preference).
- **Search History:** Automatically saves recent city searches using Local Storage.
- **Responsive Design:** Works perfectly on desktop, tablet, and mobile devices.

## 🛠️ Tech Stack

- **HTML5** - For app structure and layout.
- **CSS3** - For styling, animations, flexbox/grid layouts, and glassmorphism effects.
- **JavaScript (Vanilla)** - For DOM manipulation, API fetching, and local storage management.
- **OpenWeatherMap API** - For fetching real-time weather data.
- **Font Awesome** - For clean and scalable vector icons.
